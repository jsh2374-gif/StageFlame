StageFlame Hub Instructions

This guide explains how to use the StageFlame Hub to manage wireless flame fixtures during setup, rehearsal, and live operation.

1. Powering the System
Power on the StageFlame Hub first, then turn on all StageFlame fixtures. Connected fixtures will automatically appear in the Nodes page once detected.
2. Connecting to WiFi
Open the Network / WiFi page to connect the Hub to the venue or in-house wireless network. Once connected, the Hub can be accessed from any device on the same network using the local dashboard address.
3. Creating Groups
Use the Groups page to create fixture groups such as chandeliers, torches, sconces, or table candles. Fixtures can be assigned to groups for synchronized control and shared addressing.
4. Addressing Fixtures
Fixtures may operate in individual addressing mode or group addressing mode. DMX start addresses can be assigned from the Nodes or Groups pages.
Addressing Modes
3-Channel uses two control channels: Hero Flame intensity and Flicker intensity. This is best when you want direct control and want to use fewer lighting channels.

3-Channel adds a third control channel for flame personality. Low values select Calm, mid-low values select Light Air, mid-high values select Breeze, and high values select Storm.

3-Channel lets a full group of fixtures respond together from one address block. This is ideal for chandeliers, wall sconces, table candles, torch lines, and other grouped scenic elements.
Match Group Settings vs Individual Control
Each fixture can operate in only one control style at a time.

Match Group Settings means the fixture follows the settings assigned to its group. This is best for chandeliers, sconces, tables, or any fixtures that should behave together.

Individual Control unlocks the fixture’s own settings so it can have separate intensity, Flicker, bounce rate, low range, and high range values.

Switching one option on automatically disables the other.
Node Settings Dropdown
Each fixture has a Settings button. Open it to choose the fixture’s DMX mode and adjust the controls available for that mode.

3-Channel shows Hero Flame intensity and Flicker only.

3-Channel shows Hero Flame intensity, Flicker, and a Life Mode slider that selects Calm, Light Air, Breeze, Windy, or Storm.

Life Mode controls the automatic organic bounce behavior, including strength, speed, low range, and high range.

3-Channel lets the fixture follow its assigned group while allowing local trim adjustments.
3-Channel Layout
Channel 1 — Hero Flame: main flame intensity.

Channel 2 — Flicker: secondary Flicker intensity.

Channel 3 — Life Mode: selects automatic candle life from 1–255.
Life Mode Levels
Calm: high level 255, low level 200, slow bounce.

Light Air: high level 255, low level 150, medium bounce.

Breeze: high level 220, low level 100, moderately fast bounce.

Windy: high level 100, low level 30, moderate bounce.

Storm: high level 30, low level 1, fast bounce.
5. Hero Flame
Hero Flame controls the primary animated flame behavior. Adjust intensity, low range, high range, and bounce behavior to achieve the desired flame appearance. Bounce may be enabled or disabled using the Hero Flame Bounce toggle.
6. Flicker
Flicker controls the secondary flame shimmer and dimming level. This channel is designed for supplemental glow and subtle movement enhancement.
7. Input Sources
The Hub supports wired DMX, Art-Net, and sACN input. Input priority can be configured from the DMX / Art-Net / sACN page.
8. Calmting Modes
Show Mode is intended for live performances and active operation. Fixtures respond immediately and external lighting input takes priority.

Standby Mode reduces power consumption between shows while keeping fixtures available for quick wake and reconnection.

Deep Sleep Mode is intended for overnight shutdown, storage, and transportation. Battery usage is minimized and wireless activity is greatly reduced.

Rehearsal Mode is intended for programming, setup, and testing where manual adjustment of fixtures and scenes is preferred.
9. Blackout
The Blackout command immediately disables all flame outputs across connected fixtures.
